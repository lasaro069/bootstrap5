class = "col"

La grid de bootstrap divide la pantalla en 12 partes iguales para su uso

Se debe incluir dentro de un <div class = "row"> </div>

A cada clase 'col-' le indicamos el número de columnas que debe ocupar

La suma de columnas dentro de un class = "row" no debe superar 12

si a la class="col" no se le coloca una cantidad, bootstrap divide las doce columnas en la cantidad de 'col' del 'row' actual

ejemplo

        <div class = "row"> 
            <div class = "col" > columna1 </div>  
            <div class = "col" > columna2 </div>  
            <div class = "col" > columna3 </div>  
            <div class = "col" > columna4 </div>  
            <div class = "col" > columna5 </div>  
            <div class = "col" > columna6 </div>  
            <div class = "col" > columna7 </div>  
            <div class = "col" > columna8 </div>

            <div class = "col" > columna10 </div>  
            <div class = "col" > columna11 </div>  
            <div class = "col" > columna12 </div>  
        </div>

Si se supera la cantidad de columnas en un 'row', bootstrap desplaza las restantes debajo de las primeras

Bootstrap utiliza puntos de interrupción para adaptar los diseños a los doferentes tamaños de pantalla

Para ello, debemos incluir una clase adicional por cada 'breakpoint' que querramos adicionar al diseño, desde el mas pequeño hasta el más grande
